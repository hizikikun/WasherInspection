# 🔧 自動セットアップ機能

## 概要

このアプリケーションは、他のPCで使用する際に**システム構成を自動検出**し、**必要なライブラリを自動インストール**する機能を搭載しています。

## 機能

### 1. 自動システム検出

アプリ起動時に以下の情報を自動検出します：

- **プラットフォーム**: Windows / Linux / WSL2
- **CPU情報**: CPU名、物理コア数、論理コア数
- **メモリ情報**: 総メモリ容量（GB）
- **GPU情報**: NVIDIA GPUの有無、GPU名、VRAM容量

### 2. 自動パッケージインストール

不足しているパッケージを自動検出し、ユーザー確認後にインストールします：

#### 基本パッケージ
- `numpy` - 数値計算
- `opencv-python` - 画像処理
- `scikit-learn` - 機械学習
- `matplotlib` / `seaborn` - グラフ表示
- `pandas` - データ処理
- `PyQt5` - GUI
- `tensorflow` - 深層学習
- `psutil` - システム情報
- `pillow` - 画像処理
- `scipy` - 科学計算

#### 環境別追加パッケージ

**WSL2環境:**
- `opencv-python-headless` - WSL2用OpenCV
- CUDAライブラリ（GPU使用時）
  - `nvidia-cublas-cu12`
  - `nvidia-cudnn-cu12`
  - `nvidia-cuda-nvrtc-cu12`
  - `nvidia-cuda-runtime-cu12`

### 3. システム構成に応じた推奨設定

検出されたシステム構成に基づいて、最適な学習設定を推奨：

- **超高性能**: CPU 16コア以上、メモリ 32GB以上
- **高性能**: CPU 8コア以上、メモリ 16GB以上
- **標準**: それ以外

## 使用方法

### 初回起動時

1. アプリを起動
2. 不足しているパッケージがある場合、自動セットアップダイアログが表示されます
3. 「はい」を選択すると、必要なパッケージが自動インストールされます
4. インストール完了後、アプリを再起動

### 手動実行

コマンドラインから手動でセットアップを実行することもできます：

```bash
python scripts/auto_setup.py
```

## 動作フロー

```
アプリ起動
  ↓
システム構成を自動検出
  ├─ CPU情報
  ├─ メモリ情報
  ├─ GPU情報
  └─ プラットフォーム（Windows/Linux/WSL2）
  ↓
必要なパッケージをチェック
  ↓
不足しているパッケージがある？
  ├─ はい → ユーザーに確認ダイアログを表示
  │         ↓
  │     ユーザーが「はい」を選択
  │         ↓
  │     パッケージを自動インストール
  │         ↓
  │     完了メッセージ表示
  └─ いいえ → そのまま起動
  ↓
システム構成に応じた推奨設定を表示
```

## 注意事項

- 初回起動時や新しいPCでの使用時は、パッケージインストールに時間がかかる場合があります（数分～10分程度）
- ネットワーク接続が必要です
- 一部のパッケージのインストールに失敗した場合、手動でインストールする必要があります

## トラブルシューティング

### インストールが失敗する場合

手動でインストールしてください：

```bash
pip install numpy opencv-python scikit-learn matplotlib seaborn pandas PyQt5 tensorflow psutil pillow scipy
```

### WSL2環境でGPUを使用する場合

WSL2環境でのGPU使用には、追加のセットアップが必要です：

```bash
# WSL2環境内で実行
bash setup_wsl2_tensorflow_gpu.sh
```

詳しくは `WSL2_USAGE.md` を参照してください。
