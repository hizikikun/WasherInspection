# ワッシャー欠陥画像収集 - クイックガイド

## 簡単な使い方

### 1. ツールを起動
```bash
python inspectors/fundamental_fix_inspector.py
```

### 2. 撮影手順

#### ステップ1: ワッシャーを準備
- カメラの前にワッシャーを置く
- 照明を均一に
- 欠陥部分がはっきり見えるように

#### ステップ2: 選択モード開始
- **Mキー**を押す
- 「欠陥領域選択モード開始」と表示される

#### ステップ3: 4点をクリック
- マウスで欠陥部分を囲むように**4点をクリック**
- 順番は自由（自動的に矩形になります）
- 画面に「選択中: 4/4」と表示されたら完了

#### ステップ4: クラスを選択して保存
対応する数字キーを押す：
- **1キー**: 良品（good）
- **2キー**: 黒点（black_spot）
- **3キー**: 欠け（chipping）
- **4キー**: 傷（scratch）
- **5キー**: 凹み（dent）← 新規
- **6キー**: 歪み（distortion）← 新規

### 3. 画像の保存先

自動的に以下のフォルダに保存されます：

```
cs_AItraining_data/resin/
  ├── good/good_extra_training/
  ├── black_spot/black_spot_extra_training/
  ├── chipping/chipping_extra_training/
  ├── scratch/scratch_extra_training/
  ├── dent/dent_extra_training/          ← 新規
  └── distortion/distortion_extra_training/  ← 新規
```

## 欠陥の作り方（参考）

### 傷（Scratch）
- やすり、ドライバー、ナイフで表面に線状の傷をつける
- 様々な深さ、長さで試す

### 欠け（Chipping）
- ハンマー、プライヤーでエッジを欠かせる
- 大きさを変えて試す

### 黒点（Black Spot）
- マジック、ペン、汚れで点やシミをつける
- サイズを変えて試す

### 凹み（Dent）
- 軽く押す、重いものを載せる
- 深さを変えて試す

### 歪み（Distortion）
- 熱を加える、無理に曲げる
- 程度を変えて試す

## 推奨枚数

各欠陥タイプで**50枚以上**を目標に撮影してください。
- 様々な角度で撮影
- 様々な照明条件で撮影
- 様々な大きさ・程度の欠陥を撮影

## 注意事項

- **ESCキー**: 終了
- 四点選択後にキーを押すと保存されます
- 全体画像と切り抜き画像の両方が保存されます
