## 🎯 使い方

### 方法1: アプリケーションから（最も簡単）

1. **「🎓 学習」タブ**を開く
2. **「🔧 実行環境」**セクションで：
   - ✅ **「🚀 WSL2 GPUモードを使用」** にチェックを入れる
3. **「▶ 学習開始」**をクリック
4. **自動的にWSL2環境でGPU学習が開始されます**

### 方法2: WSL2内で直接実行

```bash
# WSL2を起動
wsl

# プロジェクトディレクトリに移動
cd /mnt/c/Users/tomoh/WasherInspection

# 仮想環境をアクティベート
source venv_wsl2/bin/activate

# 学習を実行（GPU自動使用）
python3 scripts/train_4class_sparse_ensemble.py
```
