# 🎯 使用方法ガイド

## ✅ 完了した機能

**Windowsアプリケーションから、WSL2 GPUモードとWindows CPUモードを選択して実行できるようになりました！**

## 🚀 使い方

### 1. アプリケーションを起動

```powershell
python dashboard/integrated_washer_app.py
```

### 2. 学習タブで実行環境を選択

1. **「🎓 学習」タブ**を開く
2. **「🔧 実行環境」**セクションで以下を選択：

#### 🚀 WSL2 GPUモード（推奨・高速）
- ✅ **チェックを入れる**: WSL2環境でGPU（RTX 4070）を使用して学習
- ✅ **高速**: CPUの10-50倍の速度
- ✅ **RTX 4070をフル活用**

#### 💻 Windows CPUモード
- ✅ **チェックを外す**: Windows環境でCPUを使用して学習
- ✅ **完全に機能**: すべての機能が利用可能
- ✅ **精度は同じ**: GPU/CPUで精度に差はありません

### 3. リソース設定を選択

- **🚀 クイック設定**: プリセットから選択（推奨）
- **🔧 カスタム設定**: 詳細な設定をカスタマイズ

### 4. 学習を開始

**「▶ 学習開始」ボタン**をクリック

## 📊 パフォーマンス比較

| モード | 速度 | GPU使用 | 推奨用途 |
|--------|------|---------|----------|
| WSL2 GPU | ⚡⚡⚡ 超高速 | ✅ RTX 4070 | 高速学習が必要な場合 |
| Windows CPU | ⚡ 通常 | ❌ CPUのみ | リアルタイム検査・軽量学習 |

## 💡 ヒント

- **高速学習が必要**: WSL2 GPUモードを選択
- **リアルタイム検査**: Windows CPUモードでも十分高速
- **両方使える**: 用途に応じて切り替え可能

## 🔍 確認方法

### WSL2環境が利用可能か確認

アプリ起動時に自動的に確認されます：
- ✅ **WSL2環境利用可能**: チェックボックスが有効
- ⚠ **WSL2環境未設定**: チェックボックスが無効（setup_wsl2_tensorflow_gpu.shを実行）

### GPU使用状況を確認

学習開始後、「📊 進捗・情報」タブでGPU使用率を確認できます。

**これで、Windowsアプリから簡単にGPU学習とCPU学習を切り替えられます！** 🎉
