# Change History Viewer

Gitの変更履歴をリアルタイムで確認できるGUIアプリケーションです。

## 機能

- **リアルタイム監視**: ファイル変更を自動検出
- **コミット履歴表示**: 最新20件のコミットを一覧表示
- **コミット詳細表示**: 選択したコミットの詳細情報を表示
- **差分表示**: コミットの変更内容（diff）を表示
- **変更ファイル一覧**: コミットで変更されたファイルの一覧を表示
- **GitHub連携**: ワンクリックでGitHubリポジトリを開く

## 起動方法

### 方法1: バッチファイルから起動（推奨）

1. エクスプローラーで `start_history_viewer.bat` をダブルクリック

### 方法2: VS Codeのターミナルから起動

```powershell
python change_history_viewer.py
```

### 方法3: コマンドプロンプトから起動

```cmd
cd C:\Users\西村康成\WasherInspection
python change_history_viewer.py
```

## 使い方

### 基本操作

1. **アプリ起動**: 起動すると自動的に最新のコミット履歴が表示されます
2. **自動更新**: 30秒ごとに自動的に履歴を更新します
3. **コミット選択**: リストからコミットをクリックすると、詳細情報が下部に表示されます

### ボタン機能

- **Refresh**: 手動でコミット履歴を更新
- **View Diff**: 選択したコミットの差分を新しいウィンドウで表示
- **View Changed Files**: 選択したコミットで変更されたファイルの一覧を表示
- **Open GitHub**: ブラウザでGitHubリポジトリを開く

### ステータス表示

- **緑色**: 変更なし（正常）
- **オレンジ色**: 未コミットの変更あり
- **青色**: 新しいコミットを検出

## 要件

- Python 3.6以上
- Gitがインストールされていること
- tkinter（Pythonに標準で含まれています）

## トラブルシューティング

### エラー: "git command not found"

Gitがインストールされていないか、PATHに追加されていません。
Git for Windowsをインストールしてください。

### エラー: "No module named 'tkinter'"

tkinterがインストールされていません。以下のコマンドでインストールしてください：

```powershell
pip install tk
```

### アプリが起動しない

Pythonが正しくインストールされているか確認してください：

```powershell
python --version
```

## 機能拡張

カスタマイズしたい場合は、`change_history_viewer.py` を編集してください。

- 監視間隔の変更: `monitor_changes()` 関数内の `time.sleep(30)` を変更
- 表示コミット数の変更: `get_git_commits(limit=20)` の `limit` を変更
- UIのカスタマイズ: `setup_ui()` 関数を編集

## 注意事項

- アプリを閉じると監視も停止します
- バックグラウンドで実行する場合は、ウィンドウを最小化してください
- 大量のコミットがある場合、表示に時間がかかる場合があります
